BANNERS (TPS TAGS)

Ad Sizes

 
 

Third Party Served Tags

  • Third-party ads must be in an iFrame or internal redirect formats. We do not accept Java script on our website in order to maintain security, therefore, please ensure your ad tags contain no script. Also, please ensure the tag includes an IMG SRC link and HREF link.
  • Please ensure all third party tags and tracking pixels provided are secure. All URL components of ad tags must be SSL compliant. Every URL should start with https://
  • All tags should open in a new window and have the target=”_blank” attribute.
  • A backup (standard / basic / default) baseline image and Click URL is to be submitted for each tag. This is to ensure people who do not have Frames enabled on their browser are shown the Ad, and not a blank image.
  • No Pop-Ups or Pop-Unders are allowed.
  • Expedia only accepts 3rd party ad tags from the below approved vendor list: DoubleClick, Atlas Rich Media, Sizmek (formerly MediaMind or Eyeblaster and includes EyeWonder & Unicast), Pointroll, Flashtalking.
  • If you use any of these vendors please reach out to your Account Manager: questionmarket.com (SafeCount/DynamicLogic), Ad Safe, Double Verify, Ad Gear.
  • We Cannot Accept Adara or Sojern pixels/tags.
  • Any tags outside of this list will have to go through testing which can take up to approximately 1 week to complete.
  • Any change discovered live onsite which has not gone through the proper approval process, can and will be removed from the site until all issues are resolved.
 

Standard creative specs

  • Maximum file size: 40k .jpg, 200k .gif.
  • File Types: .jpg or .gif (15s total, 3 loops max).
 

HTML 5 Ads

  • Accepted as a 3rd party tag or a Zip Bundle creative.
  • In Zip Bundle, no cross-domain scripting allowed.
  • Zip Bundle must contain a click track macro.
  • Total max. file size: 200k.
  • All Zip Bundles must go through Tag Testing for approval prior to trafficking.
 

Extra Consideration

  • Consider reflecting inclusion and diversity through messaging and content. Link here to best practices and examples
  • Do include Terms apply on banners when there is a *savings offer (Save up to xx%). For offers such as “free first checked bag” or something similar, if it truly is offered on all purchases, and not on specific flights (or rooms), then it is not necessary.
